# Git/GitHub 활용 실습

## 프로젝트 - 프로필

1. 프로젝트 시작 
    - 바탕화면에 폴더 생성
    - `README.md` 파일이 반드시 필요
    
2. GitHub
    - 원격저장소 이름은 **반드시 `username`으로!**
    
    - 원격 저장소 설정을 마무리하고 push하고 끝!

3. 프로필 확인 : https://github.com/username
4. 실습 완료 후 체크 시트에 링크 업데이트
5. 구글에 GitHub Profile Readme라고 검색해보면 다양하게 꾸미고, 설정할수있는 방법이있다.
